# Customer-Behavior-Analysis

Project Overview: 
This project focuses on understanding customer shopping behavior for a retail company. The goal is to analyze consumer data to identify trends, improve customer engagement, and optimize marketing and product strategies.

Business Question: 
How can the company use consumer shopping data to identify trends, improve customer engagement, and strengthen marketing and product strategies?

Deliverables: 
Data Preparation (Python)
  Clean and transform the raw dataset for analysis.
Data Analysis (SQL)
  Organize data into structured tables and run queries to explore customer segments, loyalty, and purchase drivers.
Visualization (Power BI)
  Build an interactive dashboard to highlight key trends and insights.
Report & Presentation
  Summarize findings and provide business recommendations.
GitHub Repository
  Store all Python scripts, SQL files, visualizations, and documentation.

Tools Used: 
  Python
  SQL
  Power BI
  GitHub
